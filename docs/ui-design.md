# Trip.com 子部门度假/跟团游产品集合页 Landing Page UI 设计文档

**版本**：v1.0  
**日期**：2026年3月  
**目标**：为 Trip.com 子部门创建 SEO 友好、高转化率的跟团度假产品集合落地页（n 个产品展示页），面向国际用户（香港、新加坡、欧洲、美国等）。  
**页面类型**：产品集合页（Category/Collection Landing Page），支持动态过滤多个跟团游套餐。  
**核心目标**：  
- SEO 流量入口（长尾关键词如“新加坡出发欧洲跟团游 2026”）  
- 高转化（过滤 → 浏览卡片 → 点击预订）  
- 国际友好（多语言、多货币、移动优先）  
- 视觉激发“度假向往”情绪，同时保持专业信任感  

---

## 1. 品牌定位与整体风格

- **风格方向**：Modern Clean + Immersive Travel（现代简洁 + 沉浸式旅行）。  
  参考 GetYourGuide / Viator / Trafalgar 等成熟旅行平台，结合 Trip.com 的专业背书。  
  避免杂乱，强调大图视觉、白空间、卡片式网格。  

- **情感基调**：兴奋、安心、便捷。让国际用户感受到“专业可靠 + 轻松探索”。  

- **响应式优先**：移动端第一（旅行用户 70%+ 来自手机），桌面平滑降级。  

---

## 2. 色系（Color Palette）

采用**清新专业旅行色系**，适合跨文化国际用户（避免过于地域化符号）。  

**主色（Primary）**：  
- #0066FF（Trip.com 蓝色调，代表信任、专业、探索）  
- 渐变：#0066FF → #00CCFF（活力海洋/天空感）  

**辅助色（Secondary/Accent）**：  
- 活力橙/金：#FF6600 或 #FFB300（用于 CTA 按钮、特惠标签、限时促销，唤起 urgency）  
- 绿色：#00CC99（环保/自然跟团元素，可选）  

**中性色**：  
- 背景：#FFFFFF（主） / #F8F9FA（浅灰区）  
- 文字：#1A1A1A（深灰，主标题） / #555555（辅助文字）  
- 边框/分割：#E5E7EB  

**文化注意**（国际用户）：
- 蓝色全球通用信任色，适合欧美/新加坡/香港用户。  
- 橙色作为 accent，避免大面积红色（部分文化中较激进）。  
- 整体高对比，确保 WCAG AA 标准（可读性）。  

**使用规则**：
- Primary 用于导航、CTA 主按钮。  
- Accent 用于价格、评分星星、Hot 标签。  
- 大量白空间 + 高清旅行大图作为视觉主角。

---

## 3. 排版（Typography）

**字体选择**（推荐 Google Fonts 或系统安全字体，确保国际加载快）：

- **标题字体（Heading）**：Inter Bold / Poppins Semi-Bold（现代 sans-serif，清晰有力）  
  - H1：48–60px（桌面），32–40px（移动）  
  - H2：32–40px  
  - H3：24–28px  

- **正文字体（Body）**：Inter Regular / System UI（-apple-system, BlinkMacSystemFont 等）  
  - 正文：16–18px（桌面），15–16px（移动）  
  - 辅助文字：14px  

**排版原则**：
- 行高 1.5–1.6，确保长文本（如行程亮点）易读。  
- 标题大写 + 适当字间距，增强视觉冲击。  
- 多语言支持：预留字体 fallback，确保中文/英文/其他欧洲语言显示一致。  
- 避免过多字体变体，最多 2–3 种字体家族。  
- 移动端：字体自动缩放 + 可点击区域 ≥44px。

---

## 4. 页面结构与内容布局（从上到下）

**整体布局**：单列流式（移动） / 带左侧过滤的网格（桌面）。使用 CSS Grid + Flexbox 实现。

### 4.1 Hero 区（首屏，高度 80–100vh）
- **背景**：全屏高清视频/动态图片（目的地景观，如欧洲阿尔卑斯或新加坡城市+海滩），叠加暗色 overlay（30–40% 不透明）。  
- **内容**（居中或左对齐）：  
  - Logo + 导航（简洁：首页 / 目的地 / 出发地 / 语言切换）  
  - H1 主标题： “香港 / 新加坡出发 · 2026 最佳欧洲跟团度假套餐” （支持动态关键词）  
  - 子标题：简短卖点 + “限时特惠”  
  - 交互搜索栏：出发地（下拉/自动补全）、日期选择、预算范围、团型（家庭/小团/情侣/冒险）  
  - 主 CTA 按钮： “立即探索套餐” （Accent 色）  
- **SEO**：H1 自然包含核心关键词。

### 4.2 产品集合网格（核心区）
- **布局**：响应式网格（桌面 3–4 列，平板 2 列，移动 1 列）。  
- **产品卡片设计**（关键转化元素）：  
  - 高清主图（16:9 或 4:3，带 hover 轻微缩放/overlay）  
  - 标签： “Hot” / “限时” / “小团限10人” / “含往返机票” （Accent 色）  
  - 标题：产品名（如 “10天经典欧洲7国跟团游”）  
  - 关键信息：时长 · 出发日期 · 评分（星星 + 人数） · 真实点评摘要  
  - 价格：大字体显示 “从 HKD 12,999 / USD 1,599 起” （支持货币切换）  
  - CTA 按钮： “查看详情 / 立即预订” （Primary 色，hover 效果）  
- **交互**：卡片 hover 阴影提升 + 心愿单图标；点击进入详情页（带 schema markup）。  
- **SEO**：每张卡片添加 structured data (Product)。

### 4.3 中间信任与辅助区
- **热门/季节推荐** 或 **对比表格**（推荐 3–4 个套餐并列对比：价格、包含项目、团大小）。  
- **用户评价轮播**：真实 Trip.com 点评，多语言支持，带头像。  
- **行程亮点或互动地图**（可选嵌入）。

### 4.4 FAQ + CTA 区
- 手风琴式 FAQ（常见问题：签证、取消政策、团费包含什么）。  
- 底部强 CTA： “还没找到心仪套餐？联系顾问” 或 newsletter 订阅。

### 4.5 Footer  
- 链接、语言/货币切换、Trip.com 子部门声明、隐私政策。

## 5. 交互与功能要求

- **动态元素**：  
  - 过滤实时更新（无刷新）。  
  - 货币/语言切换（IP 检测默认，如新加坡用户默认 SGD + 英文/简中）。  
  - 懒加载图片 + 骨架屏（提升感知速度）。  

- **动画与微交互**（适度，不抢戏）：  
  - Hero 图片轻微 parallax 或视频 autoplay muted。  
  - 卡片 hover：scale 1.03 + shadow。  
  - CTA 按钮：点击波纹或 loading 状态。  
  - 滚动触发动画（fade-in）。  

- **技术要求**：  
  - 加载速度 < 3s（Core Web Vitals 优化）。  
  - Schema Markup：Breadcrumb + Product + FAQPage。  
  - 无障碍：alt 文字、ARIA labels、键盘导航。  
  - 集成：Trip.com API 拉取实时产品数据、价格、库存。  

- **A/B 测试点**：Hero 文案、CTA 位置、过滤器布局、价格显示方式。

---

## 6. 其他设计规范

- **图标系统**：使用一致的线条图标（旅行相关：飞机、日历、用户、星级）。推荐 Feather Icons 或自定义 SVG。  
- **图片规范**：真实高清旅行照片（避免 stock 感），包含多样人群（亚洲/欧美/家庭/情侣）。图片 alt 含关键词。  
- **移动端适配**：底部固定 “查看套餐” 按钮，简化过滤流程。  
- **SEO 额外**：Meta Title/Description 动态、内部链接到主站、博客入口增加停留时间。
